<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added **Seller's Guides** report format with customizable branding and lead capture forms tailored for UK estate agents.
- Introduced live UK property data integration from multiple sources for always-current market statistics.

## Improvements

- Enhanced social sharing with one-click export to LinkedIn, Facebook, and Twitter, including auto-generated captions.
- Optimized report generation speed by `>30%` for faster client delivery.

</Update>

<Update label="2024-09-25" description="v1.2.0" tags={["bugfix", "improvement"]}>

## Bug Fixes

- Fixed sharing issues where embedded lead capture forms failed on mobile devices.
- Resolved lead capture errors during high-traffic periods, ensuring `100%` delivery of contact information.

## Improvements

- Improved Data Sheets layout for better readability on desktop and print.
- Added preview mode before finalizing reports.

</Update>

<Update label="2024-08-15" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Launched **Market Intelligence** reports with competitor analysis and price trend visualizations.
- Integrated postcode-level data filtering for hyper-local insights.

## Bug Fixes

- Corrected data source sync errors that caused outdated sold prices in reports.
- Fixed export failures for PDF formats on certain browsers.

</Update>
