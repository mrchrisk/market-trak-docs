---
title: Slash Commands
description: Quick shortcuts for common actions in the AI Assistant.
---

Slash commands are shortcuts you can type in the assistant chat to quickly trigger common actions. Type `/` in the message input to see the available commands.

## How to use

1. In the chat input, type `/`
2. A menu appears with available commands
3. Click a command or keep typing to filter the list
4. The command pre-fills your message or triggers an action

## Available commands

### Content creation

| Command | What it does |
|---|---|
| `/post` | Start creating a social media post |
| `/email` | Start drafting an email campaign |
| `/letter` | Start writing a canvassing letter |
| `/image` | Generate an AI image |

### Data and reports

| Command | What it does |
|---|---|
| `/report` | Look up or discuss a report |
| `/data` | Fetch market data for a location |

### CRM commands

These commands only appear if you have a CRM (Loop or Street) connected:

| Command | What it does |
|---|---|
| `/forsale` | Search properties for sale |
| `/sold` | Search sold properties |
| `/lettings` | Search properties to let |
| `/buyers` | Search recent buyers |
| `/sellers` | Search recent sellers |
| `/renters` | Search renter applicants |
| `/landlords` | Search landlords |

### Utility

| Command | What it does |
|---|---|
| `/help` | Get help or ask a question |
| `/clear` | Clear the current conversation |

## Tips

- You do not have to use slash commands. You can always type naturally and the assistant will understand
- Slash commands are just a faster way to start common tasks
- CRM commands are context-aware and will search your connected CRM automatically
- If a command does not appear, it may require a feature your plan does not include (e.g. CRM integration)
